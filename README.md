# Netflix Content Analysis with Python

## Overview
This project explores the Netflix Movies and TV Shows dataset using Python, Pandas, and data visualization techniques.

The goal of the project is to practice:
- data cleaning
- exploratory data analysis (EDA)
- visualization with matplotlib
- working with real-world datasets

## Project Features
- Dataset preprocessing and cleaning
- Movies vs TV Shows comparison
- Top countries producing Netflix content
- Release year trend analysis
- Visual data exploration

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab
- Jupyter Notebook

## Dataset
Dataset source:
https://www.kaggle.com/datasets/shivamb/netflix-shows

## Key Insights
- Movies are more common than TV Shows on Netflix.
- The United States produces the largest amount of Netflix content.
- Netflix content production increased significantly after 2015.
